Projekt portfolio Data Science

Czyszczenie danych dla modeli Machine Learning

Praktyczny projekt pokazujący, jak przejść od surowych danych do uporządkowanego pipeline'u ML: analiza jakości danych, czyszczenie, preprocessing, modelowanie i porównanie wyników.

Otwórz raport Quarto Zobacz kod na GitHubie

O projekcie

Projekt został przygotowany w Quarto jako raport edukacyjny i portfolio. Na przykładzie zbioru Titanic pokazuje pełny proces przygotowania danych do modelu Machine Learning — od diagnozy problemów jakości danych po pipeline preprocessingowy w scikit-learn.

Analiza jakości danych

Sprawdzenie braków danych, duplikatów, typów zmiennych, wartości odstających oraz ryzyka data leakage.

Pipeline preprocessingowy

Użycie SimpleImputer, StandardScaler, OneHotEncoder, ColumnTransformer i Pipeline ze scikit-learn.

Model przed i po czyszczeniu

Porównanie modelu bazowego z modelem po pełnym preprocessingu przy użyciu metryk klasyfikacji.

Technologie

Python Pandas NumPy Matplotlib Seaborn scikit-learn Quarto GitHub Pages

Co pokazuje ten projekt?

Ten projekt pokazuje, że potrafię nie tylko uruchomić model ML, ale też przygotować dane w sposób świadomy: wykryć problemy, uzasadnić decyzje analityczne, uniknąć data leakage i zbudować powtarzalny proces preprocessingowy.
